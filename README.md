# EEG-Schizophrenia-CNN-BiLSTM
Hybrid 1D CNN–BiLSTM model for EEG-based schizophrenia classification

This repository contains the implementation of a hybrid CNN–BiLSTM model for EEG-based schizophrenia detection.

## 🔧 Features
- 1D CNN for spatial feature extraction
- Bi-LSTM for temporal modeling
- Stratified K-Fold Cross Validation
- Evaluation metrics: Accuracy, Precision, Recall, F1-score, AUC

## 📂 Files
- model.ipynb / model.py → Main code
- requirements.txt → Dependencies

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python model.py

## 📊 Dataset
Public datasets used:
- Moscow EEG dataset
- IBIB PAN dataset

## 📌 Note
Dataset is not uploaded due to size and privacy. Please download from original sources.
